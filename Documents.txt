How to make Migration in Laravel?
commands:
	php artisan make:migration (file name here)
Send in Database command:
	php artisan migrate
check status command:
	php artisan migrate:status
rollback command:
	php artisan migrate:rollback
reset command:
	php artisan reset
delete all table data row command:
	php artisan migrate:refresh
